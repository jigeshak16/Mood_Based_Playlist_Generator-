# Mood-Based Playlist Generator

##  Project Description
This project recommends songs based on the user's mood using Machine Learning techniques.  
It analyzes input mood data and suggests suitable songs from a dataset to enhance the listening experience.

---

##  Tech Stack
- Python  
- Google Colab  
- Pandas  
- NumPy  
- Scikit-learn  

---

##  Features
- Mood-based song recommendation  
- Dataset-driven filtering  
- Simple and user-friendly logic  
- Fast and efficient predictions  

---

## Project Structure
- mood_playlist_generator.ipynb  
- Spotify tracks dataset.csv 

---

##  How to Run
1. Open the notebook in Google Colab  
2. Upload dataset (if required)  
3. Run all cells  
4. Enter mood input  
5. Get recommended songs  

---

## Example
**Input Mood:** Happy  
**Output:** List of recommended happy songs
